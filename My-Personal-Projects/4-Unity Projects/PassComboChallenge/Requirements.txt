##  Requirements

- Unity Hub
- Unity 6000.0.24f1
- TextMeshPro
- (Optional) DOTween, Cinemachine, or other plugins if used

##  How to Run

1. Clone this repo or download the project folder.
2. Open Unity Hub and add the project.
3. Use the correct Unity version to avoid compatibility issues.
4. Open the `MainScene.unity` and press Play.
